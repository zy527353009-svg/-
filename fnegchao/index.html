📖 数据口径说明

总设备数
统计周期内出现过的唯一 SN 数量，一个 SN 代表一台设备。数据来源：排除表格中 SN="all" 汇总行后的去重计数。
DAU
当日所有设备的 dau 值总和，取自表格中 SN="all" 行的 dau 列数值。反映整体设备活跃度总量。
dau ≥ 3 设备数
当日 dau 值≥3 的设备数量，定义为活跃设备。dau 为设备当日的活跃次数或使用频次。
0dau 设备数
指定周期内 dau 始终为 0 的设备数量。包括：当日 0dau、近 3 天 0dau、近 7 天 0dau 三种口径。
近 N 天 0dau
从最新日期向前推 N 天内，所有日期 dau 均为 0 的设备数。反映持续不活跃的设备情况。
活跃率
活跃设备数 (dau≥3) / 总设备数 × 100%。最新日活跃率 = 15,498 / 24,474 = 63.3%。
